# bigdata
大数据实验
WORM是做好的第一个实验！！
用PYTHON做的一只小爬虫。
具体实现：
1.从汽车之家口碑网站上找到车型口碑页，从上面抓取车友口碑内容。
2.把所有口碑文字合并。
3.统计词频。
4.去除停用词。
5.生成wordcloud。

于是，就有了下面的效果：
